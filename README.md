# SOLSWAP 🚀

Modern Solana Token Swap Demo App with futuristic Web3 UI and smooth dashboard experience.

## 🌌 Features

- Smooth Solana Swap Interface
- Modern Web3 Design
- SOL / USDC / BONK / JUP / RAY Tokens
- Responsive Mobile Layout
- Animated UI Effects
- Wallet Connect Popup
- Demo Swap Calculator
- GitHub Pages Ready

## ⚡ Tech Stack

- HTML5
- CSS3
- JavaScript
